# Practice Project

A lightweight HTML/CSS/JS prototype featuring a 3x3 container grid layout with a branded logo panel and login-style interface.

## Project Overview

Built with plain HTML, CSS, and JavaScript—no external libraries or build tools required. The project demonstrates responsive grid layouts with optional dynamic color animations.

## Version History

## Current Status

**Active Features:**
- 3x3 grid layout with styled containers
- Central logo panel with branding and flicker animation
- Login interface panel with action links
- Responsive design with CSS styling
- Menu button in top-left container

**Optional Features:**
- Dynamic random color animation (disabled by default; uncomment script tag to enable)

## Files

- **Practice.html** - Page structure with 3x3 grid layout, central brand panel, and login mockup
- **practice.css** - Styling for grid containers, panels, and animations
- **practice.js** - Generates random colors and applies them to grid containers every second (optional)
- **assets/** - Folder containing images and UI resources
  - `images/` - Logo and image assets
  - `UI/` - User interface elements

## How to Run

1. Open `Practice.html` in a web browser (directly or via a live server)
2. The page displays the styled 3x3 grid with the logo and login panel
3. To enable dynamic color animation, uncomment the `<script src="practice.js"></script>` tag in Practice.html
