# Practice Project

This project is a small HTML/CSS/JS prototype featuring a 3x3 container grid layout with a central branded logo panel and a login-style interface panel.

## Project Overview

The page is built using plain HTML, CSS, and JavaScript with no external libraries or build tools required.

## Version History

### Version 1.0
- Initial static prototype with a 3x3 grid layout.
- Central branded logo panel and login-style interface panel.
- No dynamic animation enabled by default; `practice.js` script tag is commented out.

### Version 2.0
- Updated README to document the original version and the current version.
- Clarified file roles, usage, and optional dynamic color animation.
- Retains the same lightweight HTML/CSS/JS project structure.

## Files

- Practice.html: Defines the page structure, includes the stylesheet, and sets up the 3x3 grid layout with central brand and login content.
- practice.css: Contains layout rules, panel styling, and the animated logo flicker effect.
- practice.js: Generates random colors and updates the grid containers every second. The script file is included in the project, and the script tag in Practice.html is currently commented out.

## How to Run

1. Open the project folder in a browser.
2. Launch Practice.html directly, or use a live server if you prefer.
3. The page displays the styled grid layout and login-area mockup.
4. To enable the dynamic color animation, uncomment the `practice.js` script tag in Practice.html.

## Features

- 3x3 flexbox grid layout
- Branded central logo panel with flicker animation
- Login-style panel with placeholder fields and actions
- Background and container styling via CSS
- Optional random color animation through practice.js
- Lightweight and easy to inspect
