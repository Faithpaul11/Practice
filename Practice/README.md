# Practice Project

This project is a simple interactive web page that displays a grid of colored containers. Each container changes to a new random color every second, creating a dynamic visual effect.

## Project Overview

The page is built using plain HTML, CSS, and JavaScript with no additional libraries or build tools required.

## Files

- Practice.html: Defines the page structure and includes the stylesheet and script.
- practice.css: Contains the layout and styling for the container grid.
- practice.js: Generates random colors and updates the boxes at regular intervals.

## How to Run

1. Open the project folder in a browser.
2. Launch Practice.html directly, or use a live server if you prefer.
3. The page will display the animated color-changing grid automatically.

## Features

- Simple 3x3 grid layout
- Full-page background styling
- Random color animation every second
- Lightweight and easy to understand
